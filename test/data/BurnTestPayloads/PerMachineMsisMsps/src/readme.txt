This is a set of per-machine MSIs and MSPs useful for testing chainer functionality

prodAv100.msi = Product "A" version 1.0.0
prodAv101.msi = Product "A" version 1.0.1
prodBv100.msi = Product "B" version 1.0.0
prodBv101.msi = Product "B" version 1.0.1 

PatchAv101.msp = patch targeting Product A to update it to version 1.0.1
PatchBv101.msp = patch targeting Product B to update it to version 1.0.1
PatchABv101.msp = patch targeting Products A & B to update them both to version 1.0.1
